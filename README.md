A full stack grocery app built with React (frontend) and Express (backend).

## Features
- 25 products across 8 categories
- Search and category filter
- Add to cart with quantity controls
- Checkout page with delivery and payment form
- Order confirmation screen

## Tech Stack
- **Frontend:** React + Vite
- **Backend:** Node.js + Express

## How to Run

### Backend
```bash
cd backend
npm install
node index.js
```
Backend runs on http://localhost:3000

### Frontend
```bash
cd frontend
npm install
npm run dev
```
Frontend runs on http://localhost:5173
